# Cubert

A social platform for speedcubers in training.

## Development

Clone the project:

```bash
git clone https://github.com/vimichael/cubert.git
cb cubert
```

Install dependencies:

```bash
npm install
```

Seed the project:

```bash
npm run seed:users
npm run seed:algorithms
npm run seed:user-algorithms
npm run seed:posts
```

Run the development server:

```bash
npm run dev
```
