    Slowest

    Use slowest cases

    Only your slowest cases (by average) will appear in the trainer.

    Note: You may need to reload the page if your slowest cases change.

    Groups

    select all | select none

    All

    All cases can appear in the trainer.

    Balanced

    Each selected case is guaranteed to appear equally often.

    This is good for learning and reviewing algorithms.

    Realistic

    Each selected case appears with the same probability as in a real solve.

    Due to randomness, some cases may appear very often, or rarely appear.

    Exclude Trash

    Cases in the trash will not appear.

    Use this to only practice the algorithms you want to learn.

    Include Trash

    Cases in the trash will also appear.

    It is useful to see both learned and unlearned cases to avoid mixing them up, especially if they look similar.
